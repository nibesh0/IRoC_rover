Folder descriptions
========================================================================
- bin
    Please refer to the readme file in this folder to learn how to 
    run the demo tool. 
- cfg
    Configure files for module.
- console_tester
    Source code for console test. Please refer to the readme file 
    in this folder to learn how to compile the console tool.
- doc
    This document describes the usage of Application Programming Interfaces of eSPDI lib.
- eSPDI
    This folder includ all necessnary libs and header file.
- DMPreview
    All source code for DMPreview demo tool. Please refer to the readme file 
    in this folder to learn how to compile the demo tool. 
- out
    All files generated by the tool will be stored in this folder. 

* Please refer to the readme file in the sub folder.
